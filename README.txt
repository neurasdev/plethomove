Then open index.html locally to test.

Publish with GitHub Pages
1. Create a public GitHub repo (example: plethomove-site)
2. Upload all files in this folder
3. Go to Settings → Pages
4. Set Source = Deploy from a branch
5. Set Branch = main, Folder = / (root)
6. Save

Suggested URLs for App Store Connect
- Support URL: your home page
- Privacy Policy URL: yourdomain/privacy.html

Replace support@plethomove.app if needed.
