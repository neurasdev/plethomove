User Guide

PlethoMove transfers files between nearby iPhone and Mac devices using Apple's MultipeerConnectivity framework.

To test the app:

  1. Install PlethoMove(for iOS) on iPhone, and install PlethoMove(for MacOS) on macbook.
  2. Enable Wi-Fi and Bluetooth on both devices.
  3. Start Advertising and Browsing on both devices. (It should start automatically.)
  4. Connect the devices (Click the Blue "Connect" Button. Light up green means Connected).
  5. Select photos or videos on iPhone and send them to Mac, by clicking on the Blue "Pick from Recents. (Photos & Videos)" button.
  6. Select Photos and Videos just like Apple Photos app, allows finger sliding select, with the only difference that PlethoMove shows a banner with Estimated future Export time and Transfer Time. This allows users to know how much future transfer might be so user can decide how many files to be selected. It also protects and not allowing further selecting of files if the files selected exceeds the remaining iPhone disk quota (a Red "Maximum allowable selection reached" will show on the banner). This is to prevent any future futile transfer.
  7. Click the Blue "Add" button on top right of iPhone to start the transfering.
  8. Transfer will start. On both iPhone and Macbook side, a progress bar will be shown to see how much percentage is transferred. Additionally, on iPhone side, the Remaining minutes to finished will be shown to user for good visibility.
  9. On macbook side, once the transfer is done, click on the "Files Transferred Here" button in Blue, it will take user to the folder that contains all the files that have been transferred. It usually is the Documents folder of the notebook.
  10. These files can then be moved to other desired folder, or, usb disk if plugged in to macbook, with the standard select-all and copy-and-move or copy-and-paste.

The received files will appear in the Mac Documents folder and can be opened via the "Files Transferred Here" button.

No login or account is required. The app works entirely on the local network and does not upload data to any server.

Both devices must be on the same Wi-Fi network with Bluetooth enabled. PlethoMove uses the local network to discover nearby devices and transfer files between your iPhone and Mac.
